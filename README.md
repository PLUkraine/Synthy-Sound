# Synthy Sound

## How to play
Use keyboard to play sound. The following keys are mapped: ZSXCFVGBNJMK,

## How to build
To build this app you need C++ compiler and a [SDL2 library](https://www.libsdl.org/download-2.0.php "Download link").
